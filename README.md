# MediBioDraft 

임상 데이터와 raw single-cell / bulk RNA-seq 데이터를 받아 통계·바이오인포매틱스 분석을 대행하고, 논문 초안 및 투고 방향을 제안하는 연구 분석 대행 서비스 사이트입니다. 온라인 접수와 함께 현장 방문 컨설팅도 지원합니다.

## 서비스 소개

- **임상 데이터 분석**: 코호트/후향적 임상 데이터 통계 분석, 생존분석, 다변량 회귀, 예후 모델링
- **Single-cell RNA-seq 분석**: Raw FASTQ 또는 h5ad/h5 파일 기반 QC, 클러스터링, 세포유형 주석, batch correction
- **Bulk RNA-seq 분석**: nf-core 기반 정렬·정량, 차등발현(DESeq2/edgeR), GSEA, Figure 및 논문 초안 작성
- **온/오프라인 컨설팅**: 화상 미팅은 전국, 현장 방문은 서울/수도권 지역 대상 (분석 결과 설명, 랩미팅 참석, 투고 전략 자문)

## 진행 프로세스

1. 온라인 접수 (데이터 종류, IRB 승인 여부, 연구 배경 입력)
2. 무료 데이터 검토 및 견적 안내 (1~2 영업일)
3. 계약 및 데이터 전달 (보안 채널, IRB 승인 확인)
4. 분석 수행 (QC → 통계/바이오인포 분석, 중간 진행상황 공유)
5. 논문 초안 및 방향 제안서 전달 (Figure, 초록, 타겟 저널 추천)
6. 온라인 또는 현장 컨설팅 미팅

## 개인정보 및 IRB 안내

- 접수 폼에 개인정보 수집·이용 동의(필수 체크박스)가 포함되어 있습니다.
- 임상 데이터/인체유래물을 다루는 경우 IRB(기관생명윤리위원회) 승인 여부를 접수 단계에서 확인하며, 승인 전이라도 프로토콜 자문 접수는 가능합니다.
